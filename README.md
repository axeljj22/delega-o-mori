# Delegá o Morí

Un juego incremental tipo *Universal Paperclips* sobre delegar tu negocio en IA.

Gestionás una agencia de marketing. La métrica que importa **no es la plata: son tus horas**.
Empezás haciendo cada tarea a mano, clic por clic. Vas delegando en workers de IA subiendo
los 5 niveles del framework **WORKIA** — Prompt → Asistente → Workflow → Agente → Orquestador —
hasta despedirte de tu propia operación.

**Ganás** facturando $15.000/sem trabajando ≤12h tuyas, sostenido 3 semanas.
**Perdés** por quiebra o por burnout (80h).

Las cosas se desbloquean de a poco: arrancás con un solo botón y la complejidad se gana, no se
muestra. Hay eventos que castigan el hype (el gurú del curso de prompts mágicos) y premian los
sistemas.

## Jugar

Es un único archivo estático, sin build. Abrí `index.html` en cualquier navegador.

## Stack

HTML + CSS + JavaScript vanilla. Cero dependencias.
